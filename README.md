# Global Health Disparities Analysis

## Project Overview
This project analyzes how socioeconomic factors influence life expectancy globally using World Bank data (2015-2022).

## Objectives
1. Identify key determinants of life expectancy
2. Analyze regional disparities  
3. Provide data-driven policy recommendations

## Project Structure
- scripts/ - R scripts for analysis
- data/ - Raw and processed data
- outputs/ - Visualizations and exports
- reports/ - RMarkdown reports
- viz/ - Interactive visualizations
